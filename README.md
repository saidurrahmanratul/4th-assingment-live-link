1. Difference between getElementById, getElementsByClassName, and querySelector / querySelectorAll
   getElementById selects one element by id
   getElementsByClassName selects multiple elements by class
   querySelector/querySelectorAll select elements using CSS selectors.

2. How do you create and insert a new element into the DOM?
   document.createElement() for create element
   appendChild() for insert new element.

3. What is Event Bubbling?
   Event starts from the target element and moves up to its parent elements.

4. What is Event Delegation in JavaScript? Why is it useful?
   It lets a parent handle events for its children.
   And it reduce listeners and working for dynamic elements.
